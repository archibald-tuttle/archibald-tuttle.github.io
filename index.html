Installing FFmpeg on Windows

To run these conversion tools misptools FFmpeg must be installed on your system and the installation path must be included in the Windows environment variables (PATH).

A) Install FFmpeg

If FFmpeg is not installed, you can download it from the official site: ffmpeg.org. Follow the instructions for your platform (Windows, macOS, Linux) and make sure to install the full version of FFmpeg.
  • Download a package containing binaries ffmpeg-git-full.7z from the certified developers' webpage gyan.dev. Extract and rename the extracted folder as "ffmpeg".
  • Move the folder to an appropriate location like C:\Program Files\.

    • B) Add FFmpeg Tools to Environment Variables

    If you haven't done this already, you need to add the FFmpeg path to the system environment variables. Here's how to do it on Windows:

  • Locate the directory where you extracted or installed FFmpeg. It should contain ffmpeg.exe, ffprobe.exe, and ffplay.exe.
  • Go to “Control Panel” → “System” → “Advanced System Settings” → “Environment Variables”.
  • In the “System variables” section, select “Path” and click “Edit”.
  • Add a new entry with the full path to the folder that contains ffmpeg.exe, ffprobe.exe, and ffplay.exe.
  • Click “OK” to close all windows.

    • Verify FFmpeg Installation

    Open the Command Prompt (CMD) and type:

    ffmpeg -version

    If installed correctly, you should see the version of FFmpeg. Otherwise, the command won’t be recognized.

    After confirming that ffmpeg and ffprobe are installed correctly, you are ready to use the tools!
